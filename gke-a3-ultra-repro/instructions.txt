Advanced Deployment Instructions
================================

Terraform group 'primary' was successfully created in directory gke-a3-ultra-repro/primary
To deploy, run the following commands:

terraform -chdir=gke-a3-ultra-repro/primary init
terraform -chdir=gke-a3-ultra-repro/primary validate
terraform -chdir=gke-a3-ultra-repro/primary apply

Destroying infrastructure when no longer needed
===============================================

Automated
---------

gcluster destroy gke-a3-ultra-repro

Advanced / Manual
-----------------
Infrastructure should be destroyed in reverse order of creation:

terraform -chdir=gke-a3-ultra-repro/primary destroy
